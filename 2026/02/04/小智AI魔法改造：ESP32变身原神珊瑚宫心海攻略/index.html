🛠️ 第一章：装备你的创世工坊

1.1 神器安装 - VS Code + ESP-IDF

✨ 召唤仪式步骤：

  1. 打开VS Code左侧的「魔法卷轴集市」(扩展商店)
  2. 搜索esp idf安装「Espressif IDF」插件
  3. 按下F1打开咒语输入框，输入ESP-IDF: Configure ESP-IDF extension
  4. 根据引导设置IDF路径（像配置神之眼一样仔细哦~）


底部工具栏有标示ESP-IDF的图标

⚠️ 注意事项：

  • 推荐选择「Express Install」快速安装模式
  • Python版本需≥3.7（建议3.10）
  • 安装路径不要有中文或空格！
  • 遇到网络问题可设置国内镜像：
